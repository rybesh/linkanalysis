  1. In a network with n nodes, we assign all nodes the same initial PageRank, 1 / n.
  2. We choose a number of steps, k.
  3. We then perform a sequence of k updates to the PageRank values, using the following rule for each update:
    1. Each page divides its current PageRank equally across its outgoing links and passes these equal shares to the pages it points to.
    2. If a page has no outgoing links, it passes all its current PageRank to itself.
    3. Each page updates its new PageRank to be the sum of the shares it receives.
